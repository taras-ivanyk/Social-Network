# Social-Network
A web application that allows users to create and customize profiles, publish posts, follow other users, and interact through likes and comments. Built with Django for the backend and a responsive HTML interface for the frontend. Implements authentication, user relationships, and CRUD functionality for posts and profiles.

Tech stack: Django 路 Python 路 HTML 路 SQLite 路 Bootstrap 

## Database

This project uses **SQLite** by default for simplicity and ease of setup.  
It can be easily switched to **PostgreSQL** for production environments.
